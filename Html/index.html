Bienvenido Al Sistema De Almacén

Este sistema permite gestionar el inventario de equipos y materiales del almacén, realizar préstamos de equipos, y materiales que se clasifican entre consumibles y no consumibles, y registrar novedades. Para acceder a todas las funciones, Inicie Sesión.
